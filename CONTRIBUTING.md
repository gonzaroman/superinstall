# Contributing to SuperInstall

First off, thank you for considering contributing to SuperInstall! It's people like you that make Linux a better place for everyone.

### How Can I Contribute?

* **Reporting Bugs:** If you find a bug, please open an issue and describe what happened.
* **Suggesting Enhancements:** Have an idea? Open an issue to discuss it!
* **Pull Requests:** 1. Fork the repo.
    2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
    3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
    4. Push to the branch (`git push origin feature/AmazingFeature`).
    5. Open a Pull Request.
