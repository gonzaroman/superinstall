**SuperInstall** is a powerful yet elegant all-in-one package manager designed for Debian-based distributions. Forget about terminal commands for different formats.

Just drag, drop, and install or uninstall: .deb, AppImage, Flatpak, and Snap from a single place.

**HOW INSTALL**
1. Go to the [Releases](https://github.com/gonzaroman/superinstall/releases) page.
2. Download `superinstall_1.0.1_all.deb`.
3. Install it via terminal:
   ```bash
   sudo apt install ./superinstall_1.0_all.deb
